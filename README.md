# Election-Blockchain-App
Election blockchain app - ethereum solidity web3 integration

### Prerequesites

- Run truffle local blockchain environment.
```bash
truffle
```

### install

```bash
npm install
```

### Truffle
```bash
truffle compile && truffle migrate
```
### Run App
- dev
```bash
  npm run dev
```

- production
```bash
  npm run start
```
